# YaoPkgNestjsSeaExample

This repository aims to show a minimal-working example for a NestJS app packaged to binaries using pkg.

## Installation

```shell
npm install
```

## Packaging

Run `npm run build`. After that there are three available scripts:

### PKG without SEA

For testing if the normal packaging works

```
npm run package
```

### PKG with SEA

```
npm run package:sea
```

### PKG with SEA Enhanced

```
npm run package:sea:enhanced
```
